4. Erhobene Daten und Verwendungszweck

4.1 Lokal gespeicherte Daten

Die Gartenhelfer-App speichert Ihre Daten primär lokal auf Ihrem Gerät. Dazu gehören:
- Informationen zu Ihren Gärten (Name, Layout, Pflanzenstandorte)
- Pflanzendaten (Namen, Bilder, Pflanzzeiten, Pflegehinweise)
- Pflegeaufgaben und deren Status
- API-Schlüssel (verschlüsselt gespeichert)
Diese Daten werden ausschließlich zur Bereitstellung der App-Funktionalität verwendet und verlassen Ihr Gerät nur, wenn Sie Funktionen nutzen, die externe APIs ansprechen.

4.2 Kamerazugriff

Die App benötigt Zugriff auf die Kamera, um Fotos von Pflanzen aufzunehmen, die zur Pflanzenidentifikation verwendet werden können. Die Fotos werden nur mit Ihrer ausdrücklichen Zustimmung an externe Dienste übermittelt.

5. Datenübermittlung an Dritte

5.1 Pl@ntNet API

Wenn Sie die Pflanzenidentifikationsfunktion nutzen, werden Pflanzenfotos an die Pl@ntNet API übermittelt. Pl@ntNet verwendet diese Daten zur Pflanzenidentifikation und möglicherweise zur Verbesserung ihres Dienstes. Weitere Informationen finden Sie in den Datenschutzrichtlinien von Pl@ntNet: https://plantnet.org/en/privacy-policy/

5.2 OpenAI API

Die App verwendet die OpenAI API (ChatGPT und DALL-E) für folgende Funktionen:
- Generierung von Pflegehinweisen für Pflanzen
- Erzeugung von Gartenvisualisierungen
- Erstellung von Pflegeaufgaben-Listen
- Analyse von Blühzeiten und Gartenkomposition
Bei der Nutzung dieser Funktionen werden relevante Pflanzendaten (z. B. Namen, Pflegehinweise, Standortbedingungen) an OpenAI übermittelt, das diese Daten gemäß ihrer Datenschutzrichtlinie verarbeitet. Weitere Informationen finden Sie hier: https://openai.com/privacy/

6. Speicherdauer
